# Prototipe Web Profil dan Katalog Digital "Snack Kacang Sembunyi" 

Repository ini memuat *source code* untuk situs web (*landing page*) responsif dari rintisan UMKM "Snack Kacang Sembunyi" di Dusun Krajan.

Proyek ini merupakan luaran utama dari **Program 7** dalam kegiatan Kuliah Kerja Nyata (KKN) Reguler Semester Genap 2025/2026, Lembaga Penelitian dan Pengabdian Pada Masyarakat (LPPM) Universitas Kristen Duta Wacana. Program besar KKN ini mengusung tema pemberdayaan masyarakat melalui pengembangan produk berbasis potensi lokal dan digitalisasi pemasaran.

## 📌 Tujuan Program
Website ini dirancang untuk memberikan identitas digital yang profesional, modern, dan bebas biaya pemeliharaan bagi rintisan UMKM Dusun Krajan. Konsumen akan dimudahkan dalam mengakses katalog digital hanya dengan memindai *QR Code* yang dicetak langsung pada stiker kemasan produk.

## ✨ Fitur dan Informasi Web
Situs web ini dirancang sebagai *landing page* responsif yang memuat informasi berikut:
* Profil usaha dari rintisan UMKM Dusun Krajan.
* Katalog produk "Snack Kacang Sembunyi".
* Tombol pemesanan yang terintegrasi langsung ke media sosial.

## 🛠️ Teknologi dan *Deployment*
* **HTML/CSS/JavaScript**: Digunakan untuk menyusun antarmuka (*front-end*) yang responsif.
* **GitHub Pages**: Website ini di-hosting secara publik dan gratis menggunakan layanan GitHub Pages.
* **QR Code Generator**: Digunakan untuk mengubah URL GitHub Pages menjadi *QR Code* fisik pada kemasan produk.
